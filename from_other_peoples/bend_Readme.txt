********************************************************
OBJECT BENDER INCLUDE FILE FOR PERSISTENCE OF VISION 3.x
********************************************************

Included files:

   Bend.inc - Object Bender include file for POV-Ray 3.x
   Bend.htm - Instructions for Bend.inc
   Readme.txt  - This file

   Pencil.pov, Flower.pov - Sample animations

See Bend.htm for full instructions regarding installation, usage,
copyright, legal, and contact information, etc.

The two *.pov files are sample scenes which use Bend.inc. Before
rendering these files, follow the instructions in Bend.htm to install
the include file.  The files should be rendered using the Final_Frame
INI file setting or the +KFF command line option (see the POV-Ray
documentation for more information regarding animations).
